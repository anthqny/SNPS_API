SNP Detection Platform

Simplified genomic analysis using FASTA and FASTQ sequencing data

About this platform

This system compares sequencing reads (FASTQ) against a genomic reference (FASTA) to identify Single Nucleotide Polymorphisms (SNPs).

The analysis evaluates:

  • Base mismatches
  • Allele frequency
  • Read depth
  • Quality score
  • Genotype classification
